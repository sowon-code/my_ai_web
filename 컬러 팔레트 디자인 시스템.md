## CSS 변수 정의
```css
:root {
  /* Primary Colors */
  --color-primary: #8C3B43;
  --color-primary-light: #B5555E;
  --color-primary-dark: #6E2C33;

  /* Secondary Colors */
  --color-secondary: #F6EDE8;
  --color-accent: #C97B5A;

  /* Background Colors */
  --color-bg-primary: #F6EDE8;
  --color-bg-secondary: #FBF7F4;

  /* Text Colors */
  --color-text-primary: #2B2A33;
  --color-text-secondary: #6E6A6F;
  --color-text-muted: #A8A2A6;

  /* Interactive Colors */
  --color-button-primary: #8C3B43;
  --color-button-hover: #6E2C33;
  --color-link: #C97B5A;
  --color-link-hover: #B5555E;
}
```

## 컬러 사용 가이드

### Primary Colors (`--color-primary`, `-light`, `-dark`)
- `--color-primary` (#8C3B43): 브랜드를 대표하는 짙은 버건디. 헤더 배너, 주요 CTA 버튼, 강조 타이틀 등 페이지에서 가장 시선을 끌어야 하는 요소에 사용
- `--color-primary-light` (#B5555E): 호버/포커스 상태나 보조 강조 요소에 사용해 primary보다 한 톤 가볍게 시선을 유도
- `--color-primary-dark` (#6E2C33): 클릭(active) 상태, 그림자와 맞닿는 진한 강조가 필요한 영역에 사용

### Secondary & Accent
- `--color-secondary` (#F6EDE8): 섹션 구분이나 카드 배경 등 넓은 면적에 부드럽게 깔아 톤온톤 분위기를 형성
- `--color-accent` (#C97B5A): 로즈골드 톤의 포인트 컬러. 아이콘, 배지, 일러스트 포인트 등 "은은하지만 눈에 띄어야 하는" 디테일 요소에 사용

### Background Colors
- `--color-bg-primary` (#F6EDE8): 페이지 전체 기본 배경. 따뜻하고 차분한 베이지 핑크 톤으로 브랜드 분위기를 깔아줌
- `--color-bg-secondary` (#FBF7F4): 카드, 모달, 입력창 등 콘텐츠 영역의 배경. primary 배경보다 밝게 설정해 콘텐츠를 자연스럽게 부각

### Text Colors
- `--color-text-primary` (#2B2A33): 제목, 본문 등 핵심 정보를 전달하는 텍스트. 차콜 톤으로 충분한 대비와 가독성 확보
- `--color-text-secondary` (#6E6A6F): 부제목, 설명 문구 등 본문보다 우선순위가 낮은 텍스트
- `--color-text-muted` (#A8A2A6): 캡션, 타임스탬프, placeholder 등 가장 낮은 우선순위의 보조 정보

### Interactive Colors
- `--color-button-primary` (#8C3B43): 주요 액션 버튼(제출, 구매 등)의 기본 배경색
- `--color-button-hover` (#6E2C33): 버튼 호버/active 상태에서 한 톤 어둡게 변화를 주어 클릭 가능함을 인지시킴
- `--color-link` (#C97B5A): 텍스트 링크의 기본 색상. 본문과 구분되면서도 과하지 않은 포인트
- `--color-link-hover` (#B5555E): 링크 호버 시 색을 살짝 진하고 붉게 바꿔 상호작용 피드백 제공

## 반응형 고려사항

### 다크 모드
다크 모드에서는 베이지 핑크 배경이 어두운 배경으로 전환되며 명도 대비를 재조정해야 합니다.
```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary: #1F1A1B;
    --color-bg-secondary: #2A2325;
    --color-text-primary: #F3EDE9;
    --color-text-secondary: #C9C2C5;
    --color-text-muted: #8B8588;
    --color-primary: #D9737B;
    --color-primary-light: #E89AA0;
    --color-primary-dark: #B5555E;
    --color-accent: #E0A082;
  }
}
```
- 어두운 배경 위에서는 버건디(`#8C3B43`)의 채도/명도가 묻히므로, 더 밝고 채도 높은 톤(`#D9737B`)으로 대체해 가독성과 강조 효과를 유지
- 텍스트 컬러는 명도를 반전시켜 충분한 대비(WCAG AA 기준 4.5:1 이상)를 확보

### 화면 크기별 적용
- **모바일**: 좁은 화면에서는 컬러 대비를 더 뚜렷하게 사용해 터치 영역(버튼 등)의 시인성을 높임. `--color-button-primary`와 배경 간 명도 차이를 충분히 확보
- **태블릿/데스크톱**: 넓은 영역에 `--color-bg-secondary`와 `--color-bg-primary`를 함께 사용해 레이어 구조(섹션 구분)를 시각적으로 표현
- **다크모드 + 모바일**: 작은 화면에서 어두운 배경과 밝은 강조색의 대비가 눈의 피로를 유발할 수 있으므로, `--color-primary-light` 계열을 우선 사용해 톤을 부드럽게 조정
